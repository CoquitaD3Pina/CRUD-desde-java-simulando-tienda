#  CRUD desde Java simulando tienda

Proyecto desarrollado en **Java** que simula el funcionamiento básico de una tienda tipo Oxxo mediante operaciones **CRUD**.

---

##  Descripción

Este sistema permite administrar productos dentro de una tienda simulada aplicando:

* Crear productosGIT 
* Consultar productos
* Actualizar información
* Eliminar registros

El proyecto sigue una estructura por capas para mejorar la organización del código.

---

##  Arquitectura del proyecto

El sistema está dividido en las siguientes capas:

* `model` → Entidades del sistema
* `dto` → Transferencia de datos
* `repository` → Acceso a datos
* `service` → Lógica de negocio
* `vista` → Interfaz de usuario
* `exception` → Manejo de errores
* `database` → Configuración de base de datos

---

## ⚙️ Tecnologías utilizadas

* Java
* Programación Orientada a Objetos (POO)
* Arquitectura por capas
* CRUD

---

##  Funcionalidades

✔ Alta de productos
